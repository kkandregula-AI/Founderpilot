The Thirteen

Each agent runs a dedicated system charter on the model best suited to its craft — deep reasoning on Opus, structured building on Sonnet, high-volume communication on Haiku. Charters, scopes and gates are enforced by the Orchestrator, not by trust.

Shared Company Memory

The blackboard every agent reads from and writes to — scoped by permission. This is how independent agents stay coherent: not by talking to each other directly, but through governed shared state.

Founder Approval Queue
